# أرقام شاشات تطبيق BM

هذه الأرقام ثابتة للاستخدام عند الإبلاغ عن أي تعديل أو مشكلة في التطبيق.

| الرقم | الشاشة |
|---|---|
| #S-001 | شاشة تسجيل الدخول |
| #S-002 | شاشة اختيار المبنى |
| #S-003 | شاشة الملخص / لوحة التحكم |
| #S-004 | شاشة إدارة الملاك |
| #S-005 | نافذة إضافة مالك |
| #S-006 | نافذة تعديل مالك |
| #S-007 | شاشة المصروفات |
| #S-008 | شاشة الدفعات |
| #S-009 | شاشة حساب المالك |

## ملاحظات

- أي شاشة جديدة يتم إضافتها لاحقًا تأخذ الرقم التالي مباشرة.
- عند طلب تعديل، استخدم الرقم مع وصف مختصر مثل: `#S-007 تعديل شاشة المصروفات`.
