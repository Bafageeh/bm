# BM - إدارة اتحاد الملاك

تطبيق لإدارة مبنى مكون من شقق مملوكة لأشخاص مختلفين، مع فصل كامل بين بيانات كل مبنى.

## البنية

```text
bm-api      Laravel API
bm-mobile   React Native / Expo mobile app
```

## الأدوار

- مدير التطبيق: يدير المباني والمستخدمين.
- مدير المبنى: يدير مبنى واحد أو أكثر بنفس الحساب، ويختار المبنى عند الدخول.
- المالك: يشاهد ملخصه ومصروفاته ودفعاته فقط.

## طريقة الحساب

توزيع مصروفات المبنى يكون بالتساوي على عدد الشقق:

```text
نصيب الشقة = إجمالي المصروفات / عدد الشقق
نصيب المالك = نصيب الشقة × عدد الشقق التي يملكها
رصيد المالك = مجموع دفعاته - نصيبه من المصروفات
```

## الخصوصية

كل البيانات مربوطة بـ `building_id`، لذلك لا تختلط بيانات مبنى مع مبنى آخر حتى لو كان نفس المدير يدير أكثر من مبنى.
